## Project Description
This repository contains my **Digital Literacy Project** based on the Digital Literacy course modules.  
The project focuses on improving awareness about digital tools, online safety, professional digital identity, email communication etiquette, and cybercrime prevention.

It includes 5 tasks such as infographic design, digital portfolio creation, coding practice platforms, email writing, and a cybercrime case study. All task screenshots and documents are uploaded in their respective folders.
